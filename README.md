# 🤖 Multi-Agent Debate System (AutoGen + xAI)

Проект представляет собой мультиагентную систему для проведения структурированных дебатов. Агенты обсуждают тему: **"Заменит ли ИИ программистов в ближайшие 10 лет?"**

## 🌟 Особенности
- **LLM:** Использована модель `grok-2` от xAI (Grok).
- **Фреймворк:** Microsoft AutoGen.
- **Архитектура:** 4 специализированных агента (Optimist, Skeptic, Analyst, Moderator).
- **Custom Logic:** Реализован кастомный алгоритм выбора спикера для соблюдения очереди раундов и автоматическое завершение (TERMINATE).

## 🛠 Инструкция по запуску
1. Склонируйте репозиторий.
2. Создайте виртуальное окружение: `python -m venv venv`.
3. Активируйте его: `.\venv\Scripts\activate` (Windows).
4. Установите зависимости: `pip install -r requirements.txt`.
5. Создайте файл `.env` и добавьте ваш ключ: `XAI_API_KEY=your_key_here`.
6. Запустите: `python main.py`.

## 📈 Анализ результатов
Наиболее сильную позицию в дебатах занял **Analyst**, так как он оперировал конкретными цифрами ROI и влиянием на ВВП, в то время как **Optimist** и **Skeptic** вели более философский спор. Использование Grok добавило дебатам остроты и актуальности данных.

Полный лог дебатов можно найти в папке `output/`.
